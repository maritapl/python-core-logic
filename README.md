# Python Core Logic
This repository contains small Python projects focused on fundamental programming concepts such as:
- Loops
- Dictionaries
- String processing
- Conditions
- Basic data structures
- Problem solving
These exercises were implemented as part of university practice and self-study to strengthen core Python logic skills.
## Projects Included
1. Balanced Parentheses Validator (Stack)
2. Word Frequency Counter (Dictionary)
3. Palindrome Checker (String processing)
4. Custom Stack Implementation
5. Anagram Checker
